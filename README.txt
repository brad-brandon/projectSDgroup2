## FastTrack Gym Fitness System - ReadMe  

### Installation Manual  
To set up and manage the FastTrack Gym Fitness System, follow these steps:  

1. **Server Setup**  
   - Download and install [UniServerZ](https://www.uniformserver.com/) for a local development environment.  
   - Extract the downloaded files and launch the server using the `UniController` application.  

2. **Database Setup**  
   - Use phpMyAdmin (included with UniServerZ) to create the `fasttrack_gym` database.  
   - Import the provided SQL file (`DATABASE DUMP FILE V1.0 - G02_02.sql`) into your database.  
   - Verify the table structure and data after importing.  

3. **Code Setup**  
   - Place the source code folder in the `www` directory of UniServerZ.  
   - Use [Notepad++](https://notepad-plus-plus.org/) to view or edit the project code.  
   - Update the database connection details in `config.php` to match your server setup.  

4. **Run the Website Locally**  
   - Open your browser and navigate to `http://localhost/<project-folder-name>`.  
   - Ensure all features are working correctly before hosting.
